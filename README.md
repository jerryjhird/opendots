
# open source dotfiles for fedora/ systemd / i3

## Notice:
> [!CAUTION]
> this is only for fedora/systemd, and will replace system configs and system defaults like your display manager and greeter if you do not want to replace your system configs and etc ehome/ contains all user files and configs 

### how to install
```
sh install.sh
```
(note: should be ran under your user do not run using sudo or the root account. provide password when prompted for it)

### file/folder paths:
    
~/.wallpapers/g1/, ~/.wallpapers/g2/ (wallpapers not provided)
```
drop jpg and or png files here to use as desktop wallpapers
wallpapers in either folder are cyclable through there respective keybinds

g1: super+w
g2: super+Shift+w
```

/usr/share/backgrounds/dmwp.img (not provided)
```
this image file can be any image supported by slick-greeter: [Greeter].background option
it will set the wallpaper for the login screen
```
